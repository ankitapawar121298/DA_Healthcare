# 🏥 Axon Healthcare Data Analysis Project

This project presents an end-to-end data analysis solution for Axon Healthcare using multiple tools including Excel, MySQL, Tableau, and Power BI. The goal is to analyze patient data, test results, and doctor performance to derive actionable insights.

---

## 📌 Project Overview

The dataset contains healthcare-related information such as patient details, test results (Normal, Abnormal), doctor IDs, and treatment-related data.  

This project focuses on:
- Analyzing patient distribution
- Identifying abnormal test result trends
- Evaluating doctor workload
- Comparing department performance

---

## 🛠️ Tools & Technologies Used

- **Excel** – Data cleaning, pivot tables, initial analysis  
- **MySQL** – Data querying, joins, aggregations  
- **Tableau** – Interactive dashboards and visualization  
- **Power BI** – Business intelligence reporting  

---

## 📊 Key Features & Analysis

- ✔️ Percentage of Abnormal vs Normal Results  
- ✔️ Doctor-wise Patient Load (Workload Analysis)  
- ✔️ Department-wise Performance Comparison  
- ✔️ Interactive Filters for dynamic analysis  
- ✔️ KPI Metrics for quick insights  

---

## 📈 Tableau Dashboard

The Tableau dashboard provides an interactive view of healthcare data with visualizations such as bar charts, pie charts, and KPI indicators.  

**Key Insights:**
- High percentage of abnormal cases in specific categories  
- Uneven workload distribution among doctors  
- Trends in patient test results  

---

## 📊 Power BI Dashboard

The Power BI report focuses on business-level insights with enhanced data modeling and visual storytelling.

---

## 🗄️ MySQL Analysis

- Used SQL queries to extract, filter, and aggregate healthcare data  
- Implemented joins and group by operations for deeper insights
